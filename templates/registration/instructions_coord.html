Coordinator's HowTo

The UserPortal is split into two parts for the coordinators
The Team Event Coordinators have an Equivalent Module in place
  • The Portal : Here You check the answers of various entried
  • The Admin : Here You can create and change the Questions

Viewing Entries

  1. Login with the UserName and password provided to you.
  2. Click on the view answers link in the side menu.
  3. The View Answer page contains the following :
  • All The Registered Entries
  • Besides the Entries there is a Score Assigned
  • And Indicators indicating the entries' status

4. Click on the entry you wish to read. It opens up in a box below.
5. The Page contains information reagarding whether the entry is "Read,Interesting,Selected" and a "Score".
6. Click whichever is applicable and add a score to for the entry if neccessary.
7. Important : Click The SAVE Button Your changes will get reflected above.



The Admin Portal

1. Click on the +Add link near "Questions" to Add Questions.
2. Add the content in the Box...
3. In the type : Click on MCQ/File if the question type is MCQ/requires a file-answer.
Otherwise click on Normal for an ordinary Question. If you want to display a Message, Click on the message.

4. If you want to change the Order in which the questions appear, change the preferred id so that it rearranges itself!!
5. To add choices, go to MCQOptions in the Home Page, and again Click Add.
6. Choose the Quesiton you want to add an option, and add the text to content.
The Same is also Applicable for team events and they need to go to Team_events and Team_MCQOptions for viewing necessary changes.
After creating all your questions, go to the "Events" and "TeamEvents" pages in the admin portal, and check the "Registerable" checkbox.
