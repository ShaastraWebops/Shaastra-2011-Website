In order to embed a youtube video, go to the youtube page of the video you want to use.
Then press on the "Share" button that's visible beneath the video (in youtube).
Copy the last part of the link - the last bit of text only.
For example, if the link says http://youtu.be/SomeLinkText, then copy only the SomeLinkText part.
Paste it like this: http://www.youtube.com/v/SomeLinkText in this form. As simple as that!
